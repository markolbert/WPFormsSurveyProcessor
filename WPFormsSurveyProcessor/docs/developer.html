J4JSoftware.WpFormsSurveyProcessor Developer Information

This app relies on a number of libraries I wrote and maintain. You can visit their GitHub repositories by clicking on the library name:

Library Description
J4JLogger wrapper/extension to Serilog
J4JCommandLine command line processor that works with the IConfiguration API
DependencyInjection root object resolver that works with the IHostBuilder/IHost API and Autofac
MiscellaneousUtilities various utility methods
ColorfulHelp add on to J4JCommandLine which displays formatted command line help
WpFormsSurvey importers for WPForms form definitions and survey responses

The application is structured around a series of services, each mapped to a specific configuration of command line options. Click on the links for more information about the primary services:

Service Description
HelpService displays formatted help
DocumentationService displays documentation in a browser window
MisconfigurationService displays help and then a series of error messsages
FormInfoService parses a downloaded WordPress posts file and displays a list of forms and their ids
ParseService parses downloaded WordPress posts and WPForms survey responses files and creates an Excel spreadsheet
